# Bibliothèque Articles 2026

Interface web de consultation du catalogue Arts Alu 2026.

## Fonctionnalités
- Recherche instantanée par référence ou désignation.
- 3 modes d'affichage : Grille, Compact, Liste.
- Système de favoris (étoiles) stockés localement.
- Filtrage par Fournisseur, Gamme, Série et Type.

## Installation
1. Exécuter `python extract_data.py` pour générer `data.js` (nécessite les fichiers Excel).
2. Ouvrir `index.html` dans un navigateur.
